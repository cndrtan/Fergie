# Fergie
Fergie is an app for ADHD people to help them remember their tasks by having gamified reminder to increase their motivation.

## Tech Stack
**Platform:** iOS ``15.0``<br>
**Framework:** SwiftUI, WidgetKit, WatchKit<br>
**IDE:** XCode ``v.13.4``<br>
